1 Upload to Fasthosts

2. Go to Web Hosting ‚Üí Manage ‚Üí File Manager

3. Open the htdocs folder ‚Äî this is your website root

4. Delete any placeholder files already in there (like a default index.html)

5. Click Upload and upload abpaving-final.zip

6. Right-click the zip file ‚Üí Extract Here

7. An abpaving-final/ folder will appear ‚Äî open it

8. Select all files and folders inside it and move them up into htdocs/

  3. Open FileZilla, enter those at the top bar, click Quickconnect
  4. Navigate to htdocs/ on the right panel
  5. On the left panel, open your unzipped abpaving-final folder
  6. Select everything inside and drag it across into htdocs/
